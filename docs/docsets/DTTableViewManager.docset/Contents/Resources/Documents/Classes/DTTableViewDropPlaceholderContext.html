DTTableViewDropPlaceholderContext

open class DTTableViewDropPlaceholderContext

Thin wrapper around UITableViewDropPlaceholderContext, which automates insertion of dragItems if you are using MemoryStorage. Typically, you would not create this class directly, but use DTTableViewManager.drop(_:to:with:) convenience method.
